File descriptions

Gene based information, including annotations and cross references of various IDs.


===================
Column descriptions
===================

column             description
------             -----------
geneID             [unique identifier] Identifier of the gene. Mainly inherited from GENCODE, plus the novel lncRNA knockdown targets with IDs defined by Chung's pipeline at RIKEN.
geneName           [string] Name of the gene. Mainly inherited from the "gene name" field in GENCODE, plus the novel lncRNA knockdown targets with names defined by Chung's pipeline at RIKEN.
geneType           [string] Biotypes of the gene. Mainly inherited from the "gene type" field in GENCODE, plus the novel lncRNA knockdown targets with tagged as novel gene
CAT_geneClass      [string] If mapped to FANTOM CAT, then as gene class in FANTOM CAT; If not mapped, "__na"
cntg               [string] Contig (i.e. chromosome)
geneStart          [integer] Start (i.e. leftmost) position of the gene, 1-base
geneEnd            [integer] End (rightmost) position of the gene
strnd              [+ or -] Strand
CAT_DHS_type       [string] If mapped to FANTOM CAT, then as type DNaseI hypersensitive site in FANTOM CAT, as promoters, enhancers or dyadic, based on ROADMAP DHS; If not mapped, "__na"
gapmerID           [string] If lncRNA target, IDs of the gapmers for KD. If not, "__na"
targetID           [string] If lncRNA target, ID of the target for KD. If not, "__na"
numPrmtr           [integer] Number of promoters associated with the gene
numTrnscpt         [integer] Number of transcrpts associated with the gene
prmtrID            [string] IDs of promters associated with the gene
trnscptID          [string] IDs of the transcrpts associated with the gene
HGNC_ID            [string] ID in HGNC, based on matching GENCODE ID to HGNC. If not, "__na"
HGNC_name          [string] approved name in HGNC. If not mapped, "__na"
HGNC_symbol        [string] approved symbol in HGNC. If not mapped, "__na"
HGNC_locus_group   [string] locus group annotated in HGNC. If not mapped, "__na"
HGNC_locus_type    [string] locus type annotated in HGNC. If not mapped, "__na"
HGNC_gene_family   [string] gene family annotated in HGNC. If not mapped, "__na"
alias_name         [string] alias name annotated in HGNC. If not mapped, "__na"
alias_symbol       [string] alias symbol annotated in HGNC. If not mapped, "__na"
prev_name          [string] previous name annotated in HGNC. If not mapped, "__na"
prev_symbol        [string] previous symbol annotated in HGNC. If not mapped, "__na"
entrez_ID          [string] entrez ID cross referenced in HGNC. If not mapped, "__na"
refseq_ID          [string] refSeq ID cross referenced in HGNC. If not mapped, "__na"
VEGA_ID            [string] VEGA ID cross referenced in HGNC. If not mapped, "__na"
UCSC_ID            [string] UCSC ID cross referenced in HGNC. If not mapped, "__na"
CCDS_ID            [string] CCDS ID cross referenced in HGNC. If not mapped, "__na"
uniprot_ID         [string] uniprot ID cross referenced in HGNC. If not mapped, "__na"
homeodb_ID         [string] homeodb ID cross referenced in HGNC. If not mapped, "__na"
cosmic_ID          [string] cosmic ID cross referenced in HGNC. If not mapped, "__na"
lncrnadb_ID        [string] lncRNAdb ID cross referenced in HGNC. If not mapped, "__na"
mirbase_ID         [string] mirbase ID cross referenced in HGNC. If not mapped, "__na"
snornabase_ID      [string] snornabase ID cross referenced in HGNC. If not mapped, "__na"
orphanet_ID        [string] orphanet ID cross referenced in HGNC. If not mapped, "__na"
pseudogene_org_ID  [string] pseudogene.org ID cross referenced in HGNC. If not mapped, "__na"
avg_exonNum        [flt] average number of exon per transcripts
max_exonNum        [flt] maximum number of exons in a transcript
note               [string] notes from Chung's pipeline about cross matching various IDs, warnings etc, if any
